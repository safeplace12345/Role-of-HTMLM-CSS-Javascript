What is HTML

HTML

HTML is short for Hypertext Markup Language. HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. Each page contains a series of connections to other pages called hyperlinks. Every web page you see on the Internet is written using one version of HTML code or another.

Roles

  • Provides essential structure of web content across browsers
  • Eases readability to users
  • Is used to markup content to show how it relates or links to other sources
  • Provides easy understanding to web structures to developers

What is CSS (Cascading style sheets)

CASCADING STYLE SHEETS

Short for cascading style sheets, CSS is a language used to describe reusable styles for presenting documents written in a markup language.

Roles

  • Allows web developers to alter the layout and appearance of their web pages.
  • CSS allows web designers greater control over the web pages.

What is JaVaScRiPt (.js)

JAVASCRIPT

JavaScript is a client scripting language which is used for creating web pages. It is a standalone language developed in Netscape. It is used when a webpage is to be made dynamic and add special effects on pages like rollover, roll out and many types of graphics

Other roles

  • It is mostly used by all websites for the purpose of validation.
  • JavaScript also provides the facility of creating presentations as a website. JavaScript provides RevealJS
  • Using Node JS a web server can be created. Which is an extension of javascript
